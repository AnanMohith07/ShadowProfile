# ShadowProfile – Social Footprint Analyzer

## Overview
ShadowProfile is an AI-powered web application that analyzes publicly available online data to detect privacy risks, evaluate online reputation, and provide actionable digital footprint insights through interactive dashboards and reports.

## Features
- User Authentication
- Social Media Data Analysis
- Sentiment Analysis
- Privacy Risk Detection
- Reputation Score Calculation
- Interactive Dashboard
- Report Generation

## Technologies
- Python
- Flask
- HTML/CSS/JavaScript
- Bootstrap
- MySQL
- NLTK
- spaCy
- Chart.js
